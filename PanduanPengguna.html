⚙️ Setup Sistem
Admin Setup — konfigurasi sebelum sukan bermula
⚠️ Wajib ikut urutan ini. Jangan skip langkah kerana setiap bahagian bergantung pada bahagian sebelumnya.
📋 Urutan setup yang betul
  1. Tetapan Asas — Isi nama sekolah, nama pertandingan, tarikh sukan, bilangan lorong trek (4/6/8). Ini digunakan oleh semua modul lain.
  2. Sistem Mata — Tetapkan mata Johan=5, Naib=3, Ke-3=2, Ke-4=1. Boleh guna preset MSSD, MSSM atau SUKMA.
  3. Kategori Peserta — Tambah kategori L1, L2, P1, P2 dengan had umur dan kuota. Contoh: L1 = Lelaki, umur 7–11 tahun, kuota 3 wakil.
  4. Rumah Sukan — Tambah semua rumah dengan warna identiti. ID dijana automatik contohnya RMH-MERAH.
  5. Acara Master — Tambah semua acara (100m, 200m, Lompat Jauh dll). Pilih jenis (Track/Field/Road), format, dan kategori terlibat.
  6. Pengguna — Tambah akaun Pengurus untuk setiap rumah. Pastikan pilih rumah yang betul semasa cipta akaun.
📋 Setup Dropdown Sheet
Ini memudahkan guru isi data murid terus dalam Google Sheet dengan dropdown pilihan.
  1. Pastikan Rumah Sukan sudah diisi terlebih dahulu.
  2. Pergi ke Admin Setup → Rumah Sukan.
  3. Klik butang 📋 Setup Dropdown Sheet.
  4. Sistem akan pasang dropdown darjah (1–6) dan nama_rumah dalam Google Sheet tbl_murid.
  5. Jalankan semula jika ada rumah baru ditambah.
🔄 Jana Semula Data Murid
Jalankan selepas guru selesai isi data murid dalam Google Sheet.
  1. Pergi ke Admin Setup → Rumah Sukan.
  2. Klik butang 🔄 Jana Semula Data Murid.
  3. Sistem akan isi secara automatik: id_rumah, jantina, tarikh_lahir, id_kategori.
  4. Boleh jalankan berkali-kali — hanya kolum yang kosong akan dikemas kini.
  5. Semak log hasil — berapa rekod setiap kolum dikemas kini.

🏃 Acara & Jana
Setup acara master dan jana acara pertandingan
📌 Perbezaan Acara Master vs Acara Jana
Acara Master

Template acara. Contoh: "Lari 100m". Tidak ada kategori spesifik. Dicipta sekali oleh Admin dalam Admin Setup.

Acara Jana

Acara sebenar dengan kategori. Contoh: "Lari 100m (L1)". Dijana automatik dari Master dalam Admin Ops.

📋 Langkah tambah & jana acara
  1. Admin Setup → Acara Master → klik "+ Tambah Acara".
  2. Isi: Kod (100M), Nama (Lari 100 Meter), Jenis (Track/Field/Road), Format (Heat→Final atau Terus Final).
  3. Pilih kategori terlibat: L1, L2, P1, P2 (boleh pilih lebih dari satu).
  4. Tetapkan kuota: Wakil Individu/Rumah = bilangan murid per rumah. Untuk relay, isi Pasukan/Rumah dan bilangan ahli.
  5. Admin Ops → Jana Acara → klik "Jana Semua Acara".
  6. Sistem cipta satu rekod per acara per kategori. Contoh: 100m × 4 kategori = 4 acara jana.
⚠️ Jika ada perubahan pada Acara Master, wajib padam tbl_acara_jana dan jana semula. Admin Ops → Padam Semua Acara → Jana Semula.

📋 Pendaftaran Peserta
Pengurus Rumah mendaftar murid untuk acara
🔓 Syarat sebelum pendaftaran boleh dibuka
  1. Tetapan, Kategori, Rumah Sukan dan Acara Master sudah diisi.
  2. Acara Jana sudah dibuat (Admin Ops → Jana Acara).
  3. Data murid sudah diimport dan id_rumah sudah ada (Jana Semula Data Murid).
  4. Admin aktifkan pendaftaran: Dashboard → Kawalan Sistem → togol "Pendaftaran Peserta" = ON.
📝 Cara Pengurus daftar peserta
  1. Log masuk dengan akaun Pengurus Rumah di ?page=pengurus.
  2. Pilih acara dari senarai. Klik "Daftar Peserta".
  3. Untuk acara individu: pilih murid dari senarai → klik Daftar.
  4. Untuk acara pasukan/relay: pilih murid ahli → masukkan nama pasukan (contoh: MERAH A) → klik Daftar.

🏆 Tally medal & mata
  1. Tally medal (emas, perak, gangsa) dikira automatik apabila keputusan di-PUBLISH.
  2. Mata dikira berdasarkan Tetapan Sistem Mata (lalai: Johan=5, Naib Johan=3, Ke-3=2, Ke-4=1).
  3. Jika keputusan dipadam atau diubah, PUBLISH semula untuk kemas kini tally.
  4. Paparan tally dalam: Halaman Awam → Tally Medal, dan Dashboard Admin.

➕ Tambah pengguna baru
  1. Admin Setup → Pengguna → klik "+ Tambah Pengguna".
  2. Isi username (unik, tidak boleh sama), kata laluan (min 6 aksara), nama penuh.
  3. Pilih peranan: Admin / Pengurus / Urusetia / Pencatat.
  4. Untuk peranan Pengurus — pilih rumah sukan yang berkenaan. Ini penting!
  5. Klik Simpan. Beritahu pengguna username dan kata laluan mereka.
⚠️ Pengurus yang tidak dipilihkan rumah tidak akan dapat daftar murid.
🔑 Reset kata laluan
  1. Admin Setup → Pengguna → cari pengguna berkenaan.
  2. Klik butang 🔑 (reset kata laluan).
  3. Kata laluan baru: username + 123 (contoh: cg_merah → cg_merah123).
  4. Salin kata laluan baru dan beritahu pengguna untuk tukar segera.

🔍 Debug — semak GAS Execution Log
  1. Buka Google Apps Script Editor → klik menu "Executions" (ikon jam).
  2. Cari eksekusi yang ada status merah (gagal).
  3. Klik untuk lihat log terperinci dan mesej ralat.
  4. Log bermula dengan ✅ = berjaya, ❌ = gagal.

🔄 Reset Sistem
Gunakan untuk sukan tahun baru atau permulaan semula
⚠️ BERHATI-HATI. Reset tidak boleh diundur. Buat sandaran (eksport) data dahulu jika perlu.
🟡 Soft Reset — Reset Data Operasi (Sukan Tahun Baru)
Selamat digunakan. Data murid, rumah, kategori dan acara master DIKEKALKAN.
Acara Jana Pendaftaran Individu Pendaftaran Pasukan Heat & Peserta Heat Keputusan Keputusan Akhir
  1. Admin Setup → Reset Sistem → klik "Reset Data Operasi Sahaja".
  2. Taip kata kunci RESET OPERASI untuk sahkan.
  3. Selepas reset: Admin Ops → Jana Semula Acara.
  4. Aktifkan semula pendaftaran dalam Dashboard.
🔴 Hard Reset — Reset Sepenuhnya
SEMUA data dipadam termasuk murid, rumah, kategori. Guna hanya jika mahu mulakan sistem dari kosong.
  1. Admin Setup → Reset Sistem → klik "Reset Sepenuhnya".
  2. Taip kata kunci RESET SEMUA untuk sahkan.
  3. Sistem akan muat semula secara automatik.
  4. Ikut semula semua langkah Setup dari awal.
⚠️ Akaun admin dan tetapan asas dikekalkan. Semua pengguna lain dipadam.
